Maintenance How to Update This Dashboard
✏️ Manual Update via GitHub
1
Open your GitHub repo
Go to github.com/marcT1/vorgyn-dashboard, find index.html, click pencil ✏️ to edit in-browser.
2
Find the card to update
Use Ctrl+F to search by project name. Change milestone classes (done / active / pending), icons (✓ ◐ ○), and progress %.
3
Commit → live in ~60 seconds
Write a commit message and click Commit changes. GitHub Pages auto-redeploys.
4
Export revenue data before replacing
Revenue Tracker data lives in localStorage. Click Export CSV in the Revenue tab before uploading any new file.
🤖 AI-Assisted Update via Claude
1
Export CSV first
Click Export CSV in the Revenue Tracker tab to save your sales history before uploading a new dashboard file.
2
Paste the master prompt
Claude scans all sessions since the last version, identifies new work, and outputs the complete updated index.html.
3
Upload as index.html → done
Replace the file in your GitHub repo. Pages rebuilds in ~60 seconds.
