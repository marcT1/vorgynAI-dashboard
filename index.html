2
Find the card to update

3
Commit → live in ~60 seconds
Write a commit message and click Commit changes. GitHub Pages auto-redeploys.

Export revenue data before replacing the file
Revenue Tracker data lives in localStorage. Click Export CSV before uploading a new file to preserve your records.
